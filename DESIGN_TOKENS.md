# YITE LABS — Design Tokens

## Colours

### Core
- `--ink`:        #1C0F08  — Midnight brown. Anchor. Nav, headings, CTA backgrounds.
- `--ground`:     #FAF8F3  — Warm off-white. Primary background.
- `--ground-2`:   #F3EFE6  — Deeper cream. Secondary surfaces, cards, sidebars.
- `--chalk`:      #E4DDD0  — Borders, dividers, subtle rules.
- `--ash`:        #A89880  — Body text, secondary labels, muted elements.

### Experiment colours (one per pillar)
- `--ember`:      #E8501A  — Experiments + Tools
- `--moss`:       #1A7A4A  — Products (future)
- `--denim`:      #2D4FA8  — Thinking + Writing
- `--lavender`:   #7B5EA7  — Help + Advisory
- `--harvest`:    #C8A830  — Investing

## Typography

### Display
- Font: Bebas Neue
- Use: All headlines, wordmark, section titles, experiment numbers
- Sizes: 96px hero / 56px section / 28px subsection / 20px nav
- Letter spacing: 0.03–0.08em

### Body
- Font: Space Grotesk
- Use: All body copy, descriptions, about text
- Weight: 300 (light) for body, 500 for emphasis, 700 for card titles
- Size: 15px body / 13px secondary / 11px small

### Mono
- Font: Space Mono
- Use: Labels, tags, dates, serial numbers, nav links, metadata
- Size: 10px nav / 9px labels / 8px micro
- Always uppercase, letter-spacing 0.10–0.18em

## Spacing
Base unit: 4px
Scale: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96

## Border radius
- Sharp: 0px (buttons, experiment tags)
- Subtle: 3px (buttons)
- Card: 6–8px (cards, stat blocks)
- Container: 12–14px (page sections, large cards)
- Pill: 100px (category tags)

## The experiment key
Every experiment is numbered and colour-coded:
- EXP-001, EXP-002... (zero-padded to 3 digits)
- Format: EXP-001 · [Type] · [Status]
- Status options: Live / Building / In progress / Complete / Archived

## Logo usage
- Wordmark light: use on #FAF8F3 and light grounds
- Wordmark dark: use on #1C0F08 and dark grounds
- Mark only: use as favicon, app icon, avatar, watermark
- Minimum size: 24px width for mark, 120px for full wordmark
- Clear space: equal to the height of the kite mark on all sides

## Voice
- Sentence case always (never ALL CAPS in body copy)
- First person, direct, no jargon without payoff
- Light touch — a smile is earned, not forced
- "Not a deck. Real work." not "We provide strategic advisory services."
- Experiment copy always includes: what it is, why it exists, what changed
